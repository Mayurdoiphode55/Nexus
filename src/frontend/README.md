# Frontend Setup

This directory contains the React + Vite frontend for the application.

## Prerequisites
- Node.js 20+

## Setup Instructions

1. Install Node modules:
   ```bash
   npm install
   ```

2. Start the Vite development server:
   ```bash
   npm run dev
   ```

The application will be available at http://127.0.0.1:5173.

For the backend setup and full project overview, please refer to the main `README.md` at the root of the repository.
